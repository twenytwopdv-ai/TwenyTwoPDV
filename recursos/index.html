Recursos

Descrição detalhada dos dois aplicativos incluídos na solução: o app Caixa (atendimento de vendas) e o Sistema Gestor (admin).

Caixa

App pensado para velocidade no atendimento: busca rápida de produtos, filtro por seção, checkout simplificado, leitura de códigos e múltiplas formas de pagamento.

Páginas e scripts: vendas, fechar_caixa, historico, pedidos, integração com gateways como MercadoPago (veja MERCADOPAGO.md).

Abrir página do Caixa →

Gestor

Aplicação desktop (Electron) para gestão completa: cadastro de produtos, formação de preço, fornecedores, controle de estoque, relatórios e dashboards com métricas.

Inclui scripts de migração, integração com PostgreSQL e ferramentas para verificar e criar esquema de banco automaticamente.

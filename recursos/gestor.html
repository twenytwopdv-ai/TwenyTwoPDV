App Gestor

Aplicação desktop (Electron) para administração completa do negócio: produtos, estoque, formação de preço e relatórios.

Principais funcionalidades

  • Dashboard com métricas (total de produtos, valor do estoque, produtos com estoque baixo, novos produtos do mês).
  • Cadastro completo de produtos com validações e histórico.
  • Formação de preço automática considerando impostos e margens.
  • Gestão de fornecedores e controle de entradas/saídas de estoque.
  • Relatórios detalhados e geração de gráficos.
  • Configuração e migração do banco PostgreSQL (scripts incluídos para criação de esquema).

Instalação rápida (desenvolvimento)

  1. Abra a pasta Sistema_app no seu terminal.
  2. Instale dependências: npm install
  3. Configure o banco em db_config.json e execute node run_db_schema.js para criar tabelas.
  4. Execute: npm start

Consulte o README.md na pasta Sistema_app para detalhes técnicos, testes e recomendações de integração com o app Caixa.
